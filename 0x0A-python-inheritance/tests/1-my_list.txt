==========
1-my_list
==========

Importing class
	>>> MyList = __import__('1-my_list').MyList

Basic scenary (pos, neg, zero):
	>>> my_list = MyList()
	>>> my_list.append(11)
	>>> my_list.append(1)
	>>> my_list.append(-1)
	>>> my_list.append(0)
	>>> print(my_list)
	[11, 1, -1, 0]

Sorted list:
	>>> my_list.print_sorted()
	[-1, 0, 1, 11]

Long int:
	>>> my_list = MyList()
	>>> my_list.append(0)
	>>> my_list.append(-1)
	>>> my_list.append(1)
	>>> my_list.append(999999999999)
	>>> print(my_list)
	[0, -1, 1, 999999999999]

Lon int Sorted:
	>>> my_list.print_sorted()
	[-1, 0, 1, 999999999999]

All equals:
	>>> my_list = MyList()
	>>> my_list.append(8)
	>>> my_list.append(8)
	>>> my_list.append(8)
	>>> my_list.append(8)
	>>> my_list.print_sorted()
	[8, 8, 8 ,8]

Passed an argumento to print_sorted:
	>>> my_list = MyList()
	>>> my_list.append(8)
	>>> my_list.append(8)
	>>> my_list.append(8)
	>>> my_list.append(8)
	>>> my_list.print_sorted(10)
	Traceback (most recent call last):
		...
	TypeError: print_sorted() takes 1 positional argument but 2 were given

Empty lis:
	>>> my_list = MyList()
	>>> my_list.print_sorted()
	[]
